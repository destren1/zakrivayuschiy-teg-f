# Закрывающий тег 

## Краткое описание:

<p align="Left"> Веб-страница личной рефлексии на моем пути обучения, где я делясь своим опытом, описываю пройденные этапы и извлеченные уроки, создавая пространство для самопознания и вдохновения. </p>

## Функционал:

- Возможность поставить светлую/темную тему 
- Анимации при наведении на различные элементы страницы
- Возможность поставить лайк
- Открытие модального окна

## Технологии, использованные при создании:

- Flexbox
- Grid Layout
- CSS3 (в т.ч. анимации)
- CSS variables
- Методология BEM
- JavaScript (ES6): работа с DOM, функции
- Отзывчивая верстка (mobile-first)
- Работа с Figma (pixel-perfect верстка)
